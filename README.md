# Job Listing UI

## Setup Instructions

1. Clone the repository
2. Run npm install
3. Run npm start
4. Open http://localhost:3000

## Features
- Search by job title
- Filter by location
- Filter by job type
- Sort alphabetically
- Highlight search keyword
- Responsive layout

## Assumptions
- Data is static
- Filtering is case-insensitive
- Sort applies to current filtered results
